About Page

What is Open Code Club?

  • Open Code Club is a friendly informal meetup for people interested in coding

  • Open Code Club welcomes all abilities, from beginners to professionals!

  • You can interact as much as you like - if you want to plug in headphones, ignore everyone, and work on your project, that's ok! If you want to talk to everyone that's ok too!

  • Bring your laptops, there is power and wifi available

  • Sometimes the cafe has left over cakes they give us for free!
